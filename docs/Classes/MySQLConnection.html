  • execute(query:)

    Execute the given SQL query against the database.

    Declaration

    Swift

    public func execute(query: String) -> (CMySQLResult?, [CMySQLField]?, MySQLError?)

    Parameters

    query

    valid TSQL statement to execute.

    Return Value

    Tuple consiting of an optional CMySQLResult, array of CMySQLField and MySQLError. If the query fails then an error object will be returned and CMySQLResult and [CMySQLField] will be nil. Upon success MySQLError will be nil however it is still possible for no results to be returned as some queries do not return results.

  • nextResultSet()

    Return the next result set after executing a query, this might be used when you specify a multi statement query.

      var result = connection.execute("SELECT * FROM table1; SELECT * FROM table2;")
  var row = connection.nextResult(result) // use rows from table1

  result = connection.nextResultSet()
  row = connection.nextResult(result) // use rows from table2
